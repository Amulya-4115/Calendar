# 📅 Interactive Calendar Component

An interactive, responsive calendar component built using **Next.js (App Router)** and **React**, inspired by a physical wall calendar design.

---

## 🚀 Features

### ✅ Core Features
- 📅 Dynamic calendar (real month & year)
- 🔄 Month navigation (previous / next)
- 📆 Date range selection (start → end)
- 📝 Notes per day
- 💾 Persistent storage using localStorage
- 🔴 Note indicator on calendar

---

### 🎨 UI/UX Features
- Clean wall calendar aesthetic
- Compact square date cells
- Highlight for selected range
- Highlight for today's date
- Responsive layout:
  - Desktop → side-by-side (calendar + notes)
  - Mobile → stacked layout

---

### ✨ Additional Enhancements
- Visual feedback for interactions
- Smooth and intuitive date selection
- Selected range display

---

## 🛠 Tech Stack

- **Next.js (App Router)**
- **React (useState, useEffect)**
- **CSS (inline styling / Tailwind optional)**
- **localStorage (for persistence)**
